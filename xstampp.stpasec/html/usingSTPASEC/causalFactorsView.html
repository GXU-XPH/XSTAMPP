Using the Causal Factors Table

Adding a Causal Factor

After finishing the Control Structure, each Node of the Control Structure can be used in the Causal Factors Table to add a Causal Factor to it. Press in the column labeled "Causal Factor" to do so.

Link a Vulnerability to a Causal Factor

Causal Factors can be linked to Vulnerabilities. Press in the column labeled "Vulnerability Links" to do so. It´s the same linking procedure as described for the Unsecure Control Actions Table.

Define or Link Security Constraint corresponding to a Causal Factor

New Security Constraints can be defined for Causal Factors as they can be for Unsecure Control Actions. To do so, press in the column labeled as "Security Constraint". If this Security Constraint was already defined in the Corresponding Security Constraints Table, it should not be defined one more time. Press to link to existing Security Constraints.
