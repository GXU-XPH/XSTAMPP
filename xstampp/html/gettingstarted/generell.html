XSTAMPP Project View
New Project
Save Triggers a save of the current selected project to its respective project file in the workspace
Save all Saves all projects currently opened in XSTAMPP
Open Export
Project export Opens an Export Dialog that presents all available exports of the active plugins. For more details please refere to the respective Plugin's manual
Undo If used by the currently active editor this will revert the last user Input
Redo If available for the current editor this will re-apply changes that where previously undone. Every execution of this will re-add an item to the change history of the active editor. This action is context dependent and referes to the editor that is currently active/has focus.
Change History Opens a view for displaying a change history that includes all changes done by the user that can are available for undo/redo
