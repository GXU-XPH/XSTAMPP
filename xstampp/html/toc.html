The eXtandable STAMP Platform (XSTAMPP) is a rich client platform based on eclipse which was developed to provide a base for safety analysis processes. Therefore it provides interfaces and so called extension points to enable the user to not just work with a single but with multiple safety/hazard analysis plug-ins.
