A-STPA Project Export
Project Provides a drop down list of all A-STPA projects in the current workspace. This is mandatory since the export is created for the selected project.
Company This is an optinal field to include the name of a company. This name will be included in the "Final PDF Report" and can be ignored if it is set to be not included. The Company field can also be set globally for the current workspace in the in the Export preferences (edit -> Preferences -> Export).
Logo This is an optinal field to include a logo in the "Final PDF Report" and can be ignored if it is set to be not included. The Logo can also be set globally for the current workspace in the in the Export preferences (edit -> Preferences -> Export).
Background Color This sets the color used for the background of all headers and footers in the PDF or image exports and can be ignored if those are set to be not included. The Background Color can also be set globally for the current workspace in the in the Export preferences (edit -> Preferences -> Export).
Foreground Color This sets the color used for the font color of all headers and footers in the PDF or image exports and can be ignored if those are set to be not included. The Background Color can also be set globally for the current workspace in the in the Export preferences (edit -> Preferences -> Export).
Destination This is a mandatory field to choose the file directory in which all exports choosen in the "Exported content" are exported into
Control Structure Decoration When checked the control structure images are exported using colored component borders and icons

In the bottom part of this wizard the user can choose which of the available exports should be included in the destination directory. The small icons near pdf/image/csv or Final PDF Report can be toggled by left-clicking them, between 2-4 of the following modes:
Landscape Exports of the category are exported in landscape format(in tipped A4)
Portrait Mode Exports of the category are exported in portrait format(in normal A4)
Include in Export Exports of the categoroy are added to the export
Remove from Export Exports of the categoroy are not included to the export
