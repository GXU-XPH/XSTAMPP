Control Structure with Process Model ← Causal Factors Table → Safety Constraints of Step 2
In the Causal Factors Table the Causal Factors can be analyzed and linked to all resulting Unsafe Control Actions. A-STPA supports two ways of analysing the causal factors leading to the Unsafe Control Actions:
  • Using Scenarios to define critical situations
  • Defining safety constraints directly for each hazard
The preferd way of analyzing the causal factors can be chosen in the Project Settings.

This editor also provides a filter widget to customize the Table and to enhance the readability. Therefore the table can be filtered by seven categories:
Filter Category Type Description
ALL Text Filter In this filter mode the all Component entries can be filtered by typing in a filter word. By typing in a filter those Component entries are hidden whose text defined in the control structure don't match the filter
ACTUATOR Text Filter In this filter mode all component entries for components that are not Actuators are hidden. Further a filter word can be given to filter by the text defined in the control structure
CONTROLLER Text Filter In this filter mode all component entries for components that are not Controllers are hidden. Further a filter word can be given to filter by the text defined in the control structure
CONTROLLED_PROCESS Text Filter In this filter mode all component entries for components that are not Controlled Process components are hidden. Further a filter word can be given to filter by the text defined in the control structure
SENSOR Text Filter In this filter mode all component entries for components that are not Sensors are hidden. Further a filter word can be given to filter by the text defined in the control structure

The Causal Factors Table provides seven columns which allow to define Causal Factors in each Actuator, Controller, Controlled Process or Sensor defined in the control structure.
