Adding a Process Model to a Controller of the Control Structure

Process Models can be created inside every Controller of the Control Structure. Navigate to "Control Structure with Process Model" and drag to a Controller. Process Models are described by Process Variables. Drag to a Process Model to add a Process Variable. Process Variables are described by Process Values. Drag to a Process Variable to add a Process Value to it. Controllers and Process Models can be resized to fit to its content.
