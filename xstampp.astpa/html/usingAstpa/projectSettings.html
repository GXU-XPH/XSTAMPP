A-STPA Project Settings

General | Unsafe Control Actions Table | Users

All settings edited in the Project Settings are set only for the chosen project which can be determined in the pop-up title. The Project Settings can be opened by navigating to Project->Project Settings->General
or by right clicking the project item in the project explorer and choosing Project Settings form the context menu.

The Offered choices are:
Use Causal Scenarios Use Scenarios to refine the causal factors in the project, this adds a column 'Causal Scenarios' in the Causal Factors Table that allows to define safety constraints per causal scenario
