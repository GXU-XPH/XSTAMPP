Accidents ← Hazards → Linking of Accidents and Hazards
This editor can be used to create a list of all System-level Hazards. A Hazard can than be linked to one or multiple Accidents to indicate a causal relationship.

To get a detailed description of the editors' controls see the Common Table View page.
