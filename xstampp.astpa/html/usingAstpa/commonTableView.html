Common Table View
1
 All entries currently existing in the project are listed here, except when the filter (5) is used.
  • The entry's title can be edited directly in the table by double-clicking the title cell.
  • In the Severity column the severity of each entry is displayed. By double-clicking the cell the severity can be directly editted in the Severity Settings dialog.
  • The links column contains the an ID representation of all STPA entries currently linked to the table entry.
2
 In this editor a detailed description of, or additional notes to the entry can be inserted.
3
 This section is optional and only available if the respective STPA entry can be linked to another one e.g. an Accident to a Hazard. In the link section all possible links are displayed as editable lists (e.g. Hazard Links and Safety Constraint Links).
By clicking on the + a dialog displaying all entries available for linking can be opened to insert new linkings. To remove a linking simply select the id of the linked entry from the list and click on -.
4
Adds a new entry to the project which is assigned the next available id
e.g. in the example Accidents Editor a new Accident A-3 would be added
This can be undone
If one or more entries are selected in 1 and this button is pressed a the user is prompted with a dialog to cofirm the deletion of the selected entry. Removing an entry also removes all links with it.
e.g. In the example Accidents Editor A-1 would be deleted as well as the link with H-1, H-2 and H-4
This can be undone, doing so also re-adds all previously removed links
This works just like the normal remove if all entries are selected.
e.g. In the example Editor A-1 and A-2 as well as all Hazard Links would be deleted
This can be undone, doing so will also re-add all removed links
/ With these buttons a selected entry can be moved up/down in the list (1). However this doesn't influence the id of the entry but only the order in which the entries are displayed and exported
5
 The list (1) can be filtered by typing in a word in the Filter, this will remove all entries from the list that dont't contain the filter word. This has no impact on the project data but only affects the current opened Editor view.
6
 This widget is only visible when an entry in 1 has been selected, by clicking on the the Severity Settings can be opened
