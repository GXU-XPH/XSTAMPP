Linking View

Linking Losses and Vulnerabilities

Loss and Vulnerability Mode

If Losses and Vulnerabilities were added, they can be linked to each other in the linking view. While in Loss mode, Losses can be selected from the big list in the middle of the window to show all Vulnerabilities available for linking. By clicking on the big list in the middle can be used to select a Vulnerability in order to show all Vulnerabilities available for linking. You can switch between Vulnerability and Loss mode anytime while in linking view.

Linking Procedure

Linking Vulnerabilities to an Loss will now be explained while in Loss mode: Select a Loss from the big list in the middle. Now all Vulnerabilities available for linking are presented in the list in the upper right part of the window. If Vulnerabilities have been linked to the selected Loss before, these are listed in the lower right part of the window. The Buttons labeled with Add and Remove can be used to link and unlink Vulnerabilities to Losses. Add All and Remove All can be used to link all available items or unlink all linked items.
